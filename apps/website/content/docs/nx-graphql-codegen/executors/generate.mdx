---
title: generate
description: Generates TypeScript types and other artifacts from GraphQL schemas and operations.
---

Options can be configured in project.json when defining the executor, or when invoking it. Read more about how to configure targets and executors here: https://nx.dev/reference/project-configuration#targets.

## Examples

<Tabs items={['Basic Config', 'Custom Config File']}>
  <Tab value="Basic Config">
    ```json title="apps/your-app/project.json"
    "codegen-generate": {
      "executor": "@nx-tools/nx-graphql-codegen:generate",
    }
    ```
  </Tab>
  <Tab value="Custom Config File">
    You may want to use a custom configuration file:

    ```json title="apps/your-app/project.json"
    "codegen-generate": {
      "executor": "@nx-tools/nx-graphql-codegen:generate",
      "options": {
        "config": "custom-codegen.ts",
      }
    }
    ```
  </Tab>
</Tabs>
